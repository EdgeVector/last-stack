02 What Tom memorizes — four jobs

Job 1 · Generate
Drive outcomes onto the board

Turn intent into a frontier of Kind: pr work. Never implement product code.

Engines: outcome driver (milestone or feature materialize) · groom-board · papercut-reconciler · optional signal/probe rotator entries
Job 2 · Claim
Ship code

The only product implementer. Isolated worktree → CR/PR → merged.

Engine: kanban-pickup (+ workers) · skill kanban-agent WORK · idle smart-heal only when no driving feature
Job 3 · Reconcile
Unblock merge · close board · prove product

Merged + deploy/proof gates → done. Stuck CI/CR → heal or P0. False-done forbidden.

Surfaces: pipeline-health (owns merge-babysit + daily drain) · board-reconcile (watch + closeout + validate + reaper) · feature-prove / milestone proof
Job 4 · Intake friction
Self-heal the factory inputs

Friction becomes clustered work, not board noise. Fleet stays alive overnight.

Engines: Brain papercuts only · reconciler clusters · self-upgrade · disk · one deadman/factory observer · morning REAL_HUMAN only

07 Two-week cut plan

Week 1 · Docs & registry

No behavior risk

  • Mark program-driver DROP in README + registry paused
  • Docs: pipeline-health owns babysit + drain
  • Docs: board-reconcile = watch + closeout + validate + reaper
  • Ship this page as the operator cheat sheet
  • Cap concurrent driving FO/MS (policy line)
Week 2 · Schedules

Pause duplicates

  • Pause separate merge-babysit / drain if pipeline covers
  • Single close cadence; closeout stays 0-LLM always-on
  • Merge factory + fleet-health notify into one
  • Fold rollups into one brain-sync schedule
  • Verify heartbeats green for 3+ days
Later · Engines

Portability design

  • Shared rotator / session-miner engines
  • Probes as registry entries (smoke, mini-health)
  • Delete cold prompts after a green week
  • Still: freeze novel engines without Tom go
Order of operations Pause schedules before deleting prompts. Prefer dual-run (old + new heartbeat) for one cycle on merge/close paths. Never touch primary brain to “clean up” routines.
